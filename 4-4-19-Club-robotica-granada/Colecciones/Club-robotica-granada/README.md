Blocks created in the Jedi Hardware academy.

![](https://github.com/fgcoca/Chats/blob/master/4-4-19-Club-robotica-granada/Images/club.png?raw=true)

## Install

* Download the collection: [stable](https://github.com/fgcoca/Chats/blob/master/4-4-19-Club-robotica-granada/Colecciones/Club-robotica-granada.7z)
* To work with the collection download the compressed .7z file, unzip it and proceed to its compression in .zip format, which is the one used by icestudio. This process is forced by issues of maximum file size supported in the repository.
* Install the collection: *Tools > Collections > Add*
* Load the collection: *Select > Collection*
